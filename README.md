# Danny's Diner SQL Case Study

## Project Overview
This project analyzes customer behavior for Danny's Diner using SQL.

## Dataset
The dataset contains 3 tables:

- sales
- menu
- members

## Business Questions

1. Total amount each customer spent
2. Number of visit days
3. First item purchased
4. Most purchased item
5. Points calculation

## Tools Used

- MySQL
- SQL
- GitHub

## Insights

- Ramen was the most purchased item
- Customer A spent the most money
- Loyalty program increased repeat purchases
